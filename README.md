
## Installation

- git clone
    ```
    git clone https://github.com/Hyuoo/kdt-learn-django.git

    cd kdt-learn-django
    ```
- set virtual environment
    ```
    python3 -m venv venv
    source venv/bin/activate
    ```
- install package

    `pip install -r requirements.txt`
- run server, url
    
    `python manage.py runserver`
    
    http://127.0.0.1:8000/
